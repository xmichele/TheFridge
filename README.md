## The Fridge

Installazione locale:
 
```bash
npm install
npm run dev
```

Build di produzione:

```bash
npm run build
npm run preview
```

Note rapide:
- deploy automatico su GitHub Pages tramite GitHub Actions
- base path Pages: `/TheFridge/`
- archivio ricette originale servito da `public/support/`
- PWA offline attiva; i grossi JSON dell'archivio non sono precacheati
